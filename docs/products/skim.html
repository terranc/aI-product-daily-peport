Skim（AI 邮件客户端）

轻量级开源 AI 邮件客户端，基于 Rust 构建，仅 5MB 安装包。支持离线优先、即时搜索，可选 AI 辅助（自带 API Key）。

7 访问官网💬 ChatGPT 分析

目标受众

Windows 用户中的隐私意识强的专业人士、邮件量大的工作者、厌倦臃肿邮件客户端的用户

使用场景

  • 快速搜索整个邮箱，结果即时显示（FTS5 全文搜索）
  • AI 辅助撰写邮件草稿，语气风格可学习用户历史邮件
  • 一键摘要未读邮件，智能分类优先级
  • 离线查看和管理历史邮件
  • 通过命令面板（Ctrl+K）实现键盘流操作

设计初衷

Windows 上缺少轻量级、尊重隐私的邮件客户端。作者想做一个不做臃肿功能、不收集数据、运行快速的邮件客户端，AI 作为可选附加功能。

解决什么问题

解决 Windows 上邮件客户端要么过于臃肿（Outlook）、要么功能太少的选择困境。Skim 在轻量级和智能之间取得平衡，且用自带 API Key 的方式避免 AI 订阅收费。

评分理由

Rust + Tauri 2 技术栈性能优秀，5MB 安装包很惊艳。BYOK 模式让 AI 功能灵活无锁定。但仅支持 Windows 限制了覆盖面，且缺乏日历和联系人管理。

对标竞品

Microsoft Outlook
Outlook 功能全面但臃肿，Skim 追求极简快速且注重隐私
Thunderbird
Thunderbird 功能更丰富但缺乏 AI 能力，Skim 内置 AI 辅助
Spark Mail
Spark 有 AI 功能但属于订阅制云端服务，Skim 是开源本地优先
